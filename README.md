# RUDN.Golang

Сайт-портфолио дизайнера с backend на Go.

## Что есть в проекте

- HTML/CSS из старого проекта;
- backend на Go;
- API для услуг и заявок;
- хранение данных в map;
- отдельная страница заявки вместо pop-up;
- кнопки «Назад» на дочерних страницах;
- Swagger-документация.
